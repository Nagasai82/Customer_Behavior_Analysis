# Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, from data loading and cleaning to visualization and reporting. The goal is to transform raw data into meaningful insights using Python, SQL, PostgreSQL, and Power BI.

---

## Dataset

* Dataset loaded using Python
* Explored and analyzed to understand data quality and patterns
* Cleaned by handling missing values, duplicates, and inconsistent data
* Prepared for SQL analysis and dashboard creation

---

## Tools Used

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **PostgreSQL**
* **SQL**
* **Power BI**
* **Gamma** (Presentation)
* **Microsoft PowerPoint**

---

## Project Steps

1. Load the dataset using Python.
2. Perform Exploratory Data Analysis (EDA).
3. Clean and preprocess the data.
4. Import the cleaned data into PostgreSQL.
5. Write SQL queries to answer business questions.
6. Build an interactive Power BI dashboard.
7. Prepare a project report.
8. Create a presentation using Gamma.

---

## Dashboard

The Power BI dashboard includes:

* Key Performance Indicators (KPIs)
* Trend analysis
* Category-wise comparisons
* Interactive filters and slicers
* Visual summaries for decision-making

---

## Results

* Identified important trends and patterns in the data.
* Improved data quality through preprocessing.
* Generated insights using SQL queries.
* Built an interactive dashboard for easy data visualization.
* Presented findings through a report and presentation.

---

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open and run the Jupyter Notebook or Python script.
4. Load the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI file to explore the dashboard.
7. Review the project report and presentation.
